Odin Project Guidelines


Essentials


  1. Must perform basic math (Addition,Subtraction,Multiplication,Division)
  2. An 'operate' function takes 2 numbers and one operator to return the result.
  3. Has digits, symbols, 'equals' and 'clears' buttons.
  4. There must be a display for the calculator.
  5. Clicking any button should show it on the screen and the number stored somewhere to be used.
  6. The calculator should take at any moment up to two (2) numbers to be calculated, not more.
  7. User should be able to string together several calculations, showing the ongoing result as more digits are inputed.
  8. Round large decimals so it doesn't overflow.
  9. Pressing '=' before entering all operators shouldn't cause problems.
  10. Clear should wipe all existing data, no remnants of previous calculations should exist.
  11. Doesn't allow division by zero.

Additional (Optional)


  1. It's possible to work and input floating point numbers (decimals).
  2. Calculator should know only a dot can be used for each number. '3.22.66.4' shouldn't be possible to input.
  3. Calculator is good looking.
  4. There is a backspace button to remove one digit or symbol at a time.
  5. Keyboard support.
